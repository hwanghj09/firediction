인공지능 프로젝트 1차시 프로젝트 기획서

1. 해결하고자 하는 문제
본 프로젝트에서 해결하고자 하는 문제를 구체적으로 설명하시오.
(무엇이 문제이며, 왜 해결이 필요한지 작성)

최근 지구온난화가 지속되면서 가뭄이 길어지고 비가 오지 않아 땅이 마르고 나뭇잎이 건조해져서 작은 불씨에도 큰 산불로 번지는 문제가 심각해지고 있다. 이 문제를 해결하지 않으면 삼림과 토지가 황폐화되어 이산화탄소를 산소로 바꾸는 나무가 사라져 지구온난화가 더욱 심각해질 수 있다.

3. 프로젝트 목표
본 프로젝트를 통해 궁극적으로 달성하고자 하는 목표를 작성하시오.

기상청/공공데이터포털 등에서 평균 강수량, 산불난 전적 등의 데이터를 가지와 딥러닝으로 학습시켜 산불을 예측하고 우리나라 지도에 시각적으로 보여줄 수 있는 웹사이트를 제작하여 예방할 수 있게 하는것이 목표이다. 




4. 사용할 인공지능 기술 또는 방법
프로젝트에 활용할 예정인 인공지능 기술 또는 방법을 작성하시오.
예시) 이미지 인식, 음성 인식, 자연어 처리, 추천 시스템, 기계학습 모델, 기타 등

딥러닝 모델을 사용하여 학습을 시킬 예정이다.

데이터 분석 프로젝트인 경우에만 작성
예측하거나 분류하려는 항목을 작성하시오.
예) 이탈 여부, 질병 여부 등
지역별 산불 위험도 예측


5. 예상 결과물
최종적으로 어떤 형태의 결과물을 제작할 계획인지 작성하시오.
예시) 웹사이트, 앱, 게임, 챗봇, 이미지 분류, 기타 등

웹사이트로 우리나라 지도 상에 예측한 데이터를 토대로 산불 위험도 예측을 시각적으로 표시
초록 : 안전, 노랑 : 경고, 주황 : 위험, 빨강 : 매우 위험



사용자가 산불 위험도를 간단하게 시각적으로 알 수 있다.
정확도가 0.5를 넘어간다
오류없이 실행된다.
누구나 접속할 수 있도록 배포한다.




웹사이트에 우리나라의 지도를 표시하고 지도의 지역마다 색깔로 산불 위험도 예측한 결과로 표시한다.






7. 예상되는 어려움 또는 한계
현재 계획한 프로젝트를 수행할 때 예상되는 어려움이나 한계를 작성하시오.
예시) 데이터 확보의 어려움, 기술 구현의 어려움, 시간 부족, 성능 부족,사용자 테스트의 어려움

데이터 확보를 할 때 로그인/회원가입이 필요한 사이트가 있다. 이 회원가입을 할 때 휴대폰 인증을 할 수도 있어 어려움이 있을 것 같다.
데이터가 부족하여 예측 성능이 안좋을거같다.










8. 예상되는 사회적 영향 또는 윤리적 고려사항
본 프로젝트가 실제로 사용될 경우 예상되는 긍정적 영향 또는 부정적 영향을 작성하시오.
예시) 생활 편의 향상, 학습 도움 오분류로 인한 피해, 잘못된 정보 제공, 개인정보 문제, 기타 등

실제로 사용될 경우 산불을 미리 예방하여 산불 피해를 줄일 수 있다.
예측이 잘못되어 산불이 안났을 경우 예방을 위해 쓴 예산이 아까울 수 있다.

2~3차시에 어떤 순서로 프로젝트를 진행할 것인지 간단히 작성하시오.
예시) 자료 수집 → 구현 → 테스트 → 수정,         기획 → 개발 → 테스트 → 개선
